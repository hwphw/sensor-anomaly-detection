# Sensor Anomaly Detection 

이 프로젝트는 제조/IoT 환경에서 발생하는 센서 데이터를 분석하여 
이상 패턴을 탐지하는 머신러닝 모델을 개발하는 것을 목표로 합니다. 


---

## 프로젝트 목표 (초기 버전)
- 센서 데이터 EDA 및 시각화 
- IsolationForest를 활용한 베이스라인 모델 구축 
- AutoEncoder 등 고급 모델 실험 예정 
- FastAPI를 통한 모델 추론 API 구현 
- Docker 기반 배포까지 확장 예정
